快捷键

https://blog.csdn.net/qq_38963960/article/details/89552704
ctrl + / 注释

标签名字 + table 自动补全标签必填项 例：h1 + table 补全成闭合标签

选中代码块+Tab 整体右移

选中代码块+shift+Tab 整体左移

ctrl + o 继承类/接口后，出现可以重写的方法，ctrl多选

Alt + Insert 代码自动生成，如生成对象的 set / get 方法，构造函数，toString() 等

ALT + F12 在idea里面打开terminal(cmd的终端)，会自动定位到当前项目目录

Ctrl+Alt+I 将选中的代码进行自动缩进编排，这个功能在编辑 JSP 文件时也可以工作
